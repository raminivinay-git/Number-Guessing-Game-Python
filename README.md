# 🎯 Number Guessing Game (Python)

A simple command-line Number Guessing Game built using Python.

The program randomly selects a number between **1 and 100**, and the player has to guess it. After each guess, the program provides hints whether the guess is **too high** or **too low** until the correct number is found.

---

## 📌 Features

- Random number generation
- User-friendly input validation
- Hints for every incorrect guess
- Counts the total number of guesses
- Prevents invalid inputs
- Beginner-friendly Python project

---

## 🛠️ Technologies Used

- Python 3
- `random` module

---

## ▶️ How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/Number-Guessing-Game-Python.git
   ```

2. Navigate to the project folder:
   ```bash
   cd Number-Guessing-Game-Python
   ```

3. Run the program:
   ```bash
   python Number-Guessing-Game.py
   ```
